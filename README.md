# Enya Lite

Hackathon throwaway: an AI-personalized K-12 EAL tutor. Generates a different
text lesson, video quiz, voice conversation, and story game for every student
— same topic, different vocabulary, different cultural framing, different
illustrations.

> See `ultraplan-00-architecture.md` for the locked architecture decisions and
> `OVERNIGHT-REPORT.md` for the current build status.

## Quickstart

```bash
npm install
cp .env.example .env.local           # paste API keys
npm run dev                          # http://localhost:3000

# Optional: PDF parsing sidecar
cd docling-sidecar && python3 -m venv .venv && source .venv/bin/activate
pip install -r requirements.txt
cd .. && npm run sidecar             # http://localhost:8000
```

The app **runs without any API keys** — all live integrations gracefully fall
back to seed data + keyword-dispatched stub responses. Add keys to enable real
LLM/image/voice/YouTube calls.

## Demo path

1. **Homepage** → click "Teacher" pill (top right)
2. **Teacher chat** → click suggestion chip "Audit my textbook for Bloom's"
3. **Audit card** renders with 5 score rings + 4 prioritized recommendations
4. Click "Maya" pill → **Maya's pink butterfly dashboard**
5. "Continue Learning" → **text lesson** with butterfly metaphors at A1 vocab
6. Click "Watch" tab → **YouTube + overlay quiz** auto-pauses at 30s
7. Click "Speak" → **scripted voice tutor activity** with mic orb + transcript
8. Click "Apply" → **branching story game**: butterfly garden adventure
9. Switch to "Liam" pill → **same lesson, space-station theme, B1 vocab**
10. Terminal: `npm run demo:l4` prints **70.8% Levenshtein delta** as proof

## Stack

Next.js 16.2.1 (App Router) · React 19 · TypeScript · Tailwind 4 ·
shadcn/ui (zinc base, New York style) · Radix UI · Zustand · SWR ·
react-hook-form · Zod · motion · sonner · react-youtube · recharts ·
react-markdown · canvas-confetti · OpenAI SDK · ElevenLabs JS+React SDKs

Backend: Next.js API routes (no separate server). Backboard.io for AI
orchestration. Docling Python sidecar for PDF parsing. ElevenLabs
Conversational AI for voice. DALL-E 3 for illustrations. YouTube Data API v3
for video search. **No database** — Backboard memory + Zustand client store
+ JSON seed files only.

## Demo presenter cheat sheet

If anything fails mid-demo, type into the browser console:

```js
window.localStorage.setItem('USE_SEED_FALLBACK', 'true'); location.reload();
```

The entire demo runs from pre-generated seed JSON. Nothing is fake — it was
all generated by real LLMs in `R-03` and saved to `public/seed/`.
